# PiliPlus 加 YouTube 源

> 草稿 · 自审用 · 原则: 不把简单问题复杂化 —— 要做的就是把 YouTube 接进 PiliPlus,不多不少。

## 想做的

把 YouTube 作为第二个视频源接入 PiliPlus。播放器、过滤规则、设置 UI 全部复用,B 站现有功能不动。

## 已定的事

- **UI**: 搜索结果混排,卡片带来源角标
- **节奏**: 先 MVP(搜索 + 播放),后续按需补齐评论、字幕、订阅等

## 还没定的事

- 过滤阈值(短视频时长 / 最大播放量 / 屏蔽分区)是跨源共用一套,还是 B 站 / YouTube 各管各
